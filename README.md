# Simple Stock Report App (KR)

이 저장소는 한국 증시 주요 섹터별 대장주의 주가 변동성을 자동으로 리포트하는 파이썬 프로그램을 포함하고 있습니다.

## 주요 기능

1. **한국 증시 대장주 분석**: 삼성전자, 현대차, NAVER 등 각 섹터를 대표하는 10개 종목을 분석합니다.
2. **최근 7일 변동성 리포트**: 최근 일주일간의 수익률과 주가 변동성(최고가-최저가 대비)을 계산합니다.
3. **자동 마크다운 생성**: 실행 시 `reports/` 폴더에 날짜별 마크다운 리포트를 생성합니다.
4. **자동 업데이트**: 매주 평일(월-금) 오후 6시에 자동으로 실행되어 GitHub에 최신 리포트를 업로드합니다.

## 포함된 프로그램

- `stock_report.py`: 주가 데이터를 가져와서 분석하고 리포트를 생성하는 메인 프로그램입니다.

## 실행 방법

필요한 라이브러리 설치:
```bash
pip install yfinance pandas requests tabulate
```

리포트 실행:
```bash
python stock_report.py
```

## 생성된 리포트 확인
`reports/` 폴더에서 날짜별로 생성된 마크다운 파일을 확인할 수 있습니다.
