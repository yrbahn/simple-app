# Simple Stock Report App

이 저장소는 주요 섹터별 대장주의 주가 변동성을 리포트하는 간단한 파이썬 프로그램을 포함하고 있습니다.

## 포함된 프로그램

1. `hello.py`: 기본적인 인사 프로그램
2. `stock_report.py`: 주요 섹터별 대장주(AAPL, GOOGL, AMZN 등)의 최근 7일간 주가 변동성 및 수익률 리포트 생성

## 실행 방법

필요한 라이브러리 설치:
```bash
pip install yfinance pandas requests
```

리포트 실행:
```bash
python stock_report.py
```
